## 📌 Описание

FitLos — это веб-приложение для автоматизации анализа клиентских диалогов и поддержки администраторов фитнес-клуба «Лос Айленд». Система интегрируется с мессенджером MAX (источник диалогов) и CRM Yclients (данные о клиентах), использует **GigaChat API** и технологию **RAG** (Retrieval-Augmented Generation) для интеллектуальной обработки запросов.

**Ключевые возможности:**
- 📊 **Дашборд администратора** — три вкладки: Клиенты, Чаты, Аналитика.
- 🧠 **Анализ диалогов через GigaChat** — распознавание намерений, тональность, генерация черновиков ответов.
- 📚 **RAG-механизм** — поиск релевантных фрагментов в базе знаний клуба (прайсы, скрипты, правила) и подстановка в промпт.
- 📈 **Прогнозирование оттока (Churn Rate)** — автоматический расчёт риска ухода клиента на основе истории визитов, частоты посещений и суммы трат.
- 🔄 **Синхронизация с Yclients** — асинхронная загрузка клиентов через API.
- 🔐 **Аутентификация** — JWT-токены, хэширование паролей (sha256_crypt), формы входа/регистрации/сброса пароля.
- 📱 **Интерфейс** — адаптивный веб-дашборд на HTML + CSS (без фреймворков), шаблоны Jinja2.
